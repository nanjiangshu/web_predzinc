This folder contains the dataset used for evaluation of PredZinc paper (Shu
et. al., 2008)

Description of the dataset

1. File "passe.fasta"
   FASTA file containing the amino acid sequences

2. File "passe.seqid.txt"
   Text file with the sequence IDs

3. File "revised-closeMetalPro.dat"
   Definition of metal-binding sites extracted from the PDB

4. Folder "5-fold-cross-validation"
   Containing sequence IDs for the 5-fold cross-validation

5. Folder "profiles"
   Containing sequence profiles for the dataset
